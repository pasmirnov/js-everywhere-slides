Ты что несешь вообще?

  • Node.js - среда выполнения, основанная на JavaScript движке Chrome v8.
  • Работает в одном потоке.
  • Параллелит с помощью событийно-ориентированной модели (event loop).
  • Использует неблокируещий ввод-вывод.
  • Есть свой менеджер пакетов - Node Package Manager, npm.

Как дела у ноды

  • Уже 7 лет (с 2009, npm since 2010).
  • v7.9.x (v6.10.2 LTS)
  • 436k модулей в npm (~1100 каждый день)
  • Народ использует: Netflix, Uber, NASA, PayPal, LinkedIn и еще куча
